444 /*
445  *-- The structures are determined by SYMBFACT and used by DDISTRIBUTE.
446  *
447  * (xlsub,lsub): lsub[*] contains the compressed subscript of
448  * rectangular supernodes; xlsub[j] points to the starting
449  * location of the j-th column in lsub[*]. Note that xlsub
450  * is indexed by column.
451  * Storage: original row subscripts
452  *
453  * During the course of sparse LU factorization, we also use
454  * (xlsub,lsub) for the purpose of symmetric pruning. For each
455  * supernode {s,s+1,...,t=s+r} with first column s and last
456  * column t, the subscript set
457  * lsub[j], j=xlsub[s], .., xlsub[s+1]-1
458  * is the structure of column s (i.e. structure of this supernode).
459  * It is used for the storage of numerical values.
460  * Furthermore,
461  * lsub[j], j=xlsub[t], .., xlsub[t+1]-1
462  * is the structure of the last column t of this supernode.
463  * It is for the purpose of symmetric pruning. Therefore, the
464  * structural subscripts can be rearranged without making physical
465  * interchanges among the numerical values.
466  *
467  * However, if the supernode has only one column, then we
468  * only keep one set of subscripts. For any subscript interchange
469  * performed, similar interchange must be done on the numerical
470  * values.
471  *
472  * The last column structures (for pruning) will be removed
473  * after the numercial LU factorization phase.
474  *
475  * (xusub,usub): xusub[i] points to the starting location of column i
476  * in usub[]. For each U-segment, only the row index of first nonzero
477  * is stored in usub[].
478  *
479  * Each U column consists of a number of full segments. Each full segment
480  * starts from a leading nonzero, running up to the supernode (block)
481  * boundary. (Recall that the column-wise supernode partition is also
482  * imposed on the rows.) Because the segment is full, we don't store all
483  * the row indices. Instead, only the leading nonzero index is stored.
484  * The rest can be found together with xsup/supno pair.
485  * For example,
486  * usub[xsub[j+1]] - usub[xsub[j]] = number of segments in column j.
487  * for any i in usub[],
488  * supno[i] = block number in which i belongs to
489  * xsup[supno[i]+1] = first row of the next block
490  * The nonzeros of this segment are:
491  * i, i+1 ... xsup[supno[i]+1]-1 (only i is stored in usub[])
492  *
493  */
